== Description ==

Add shortcode to display currency converter form from Google.

Feature:

- Added shortcode to display form
- Get retsult from Google

== Installation ==

1. Unzip the naixer-currency-converter.zip
2. Copy naixer-currency-converter folder to wp-content/plugins
3. Go to Plugins/Installed Plugins, find Naixer Currency Converter and click Active
4. Enjoy!
